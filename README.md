#### Prerequisites

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org/)

Once these are installed, clone this repository to your file system, then run the following commands in your terminal:

```
cd uv-hello-world
npm install
npm start
```

Then browse to `localhost:5000`.
